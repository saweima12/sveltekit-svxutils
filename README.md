# SvxUtils

An easily & useful markodwn content provider for SvelteKit. use [MDsveX](https://github.com/pngwn/MDsveX) as preproocess.

## Features

- Markdown source provide
- Classification

## Install

```bash
npm install -D sveltekit-svxutils
# or yarn add -D sveltekit-svxutils
```
